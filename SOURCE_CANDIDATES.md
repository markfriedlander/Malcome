# Source Candidates

This is not an implementation list.

It is a living map of source directions that fit Malcome's doctrine:
places where culture appears before mainstream validation and where attention means something.

Use [SOURCE_DOCTRINE.md](/Users/markfriedlander/Desktop/Fun/Malcome/SOURCE_DOCTRINE.md) as the rulebook when evaluating any candidate here.

Each candidate should eventually earn a short justification in these terms:
- why it is early
- why it is selective
- why its attention matters
- how it could corroborate the existing network

## Selection Questions

Before adding a source, ask:

Does it tend to surface creators, scenes, ideas, aesthetics, or communities before broad validation?

Is it a meaningful selector, validator, connector, or scene bridge rather than just another publisher?

Does it add a genuinely different cultural vantage point instead of duplicating a family already in the network?

Does it fit an existing adapter cleanly, or is the cultural value high enough to justify new parser work?

Can Malcome explain why this source is predictive if it later starts weighting it more heavily?
