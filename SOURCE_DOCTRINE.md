# Source Doctrine

Malcome is not looking for respectable coverage.
Malcome is looking for early consequential filters.

## The Rule

A good Malcome source is:
- early
- selective
- scene-embedded
- culturally consequential
- likely to notice movement before consensus forms

A weak Malcome source is:
- merely prestigious
- broadly respectable
- generic commentary
- raw event inventory
- open public firehose
- easy to ingest but low in cultural consequence

## The Mental Model

Malcome should care about sources that function like:
- selectors
- validators
- connectors
- bridge figures
- scene infrastructure

Not just publishers.

## Why This Matters

The right source is not the one that already won history.
The right source is the one that is good at being early.

That often means fringe, niche, local, subcultural, or weird sources matter more than polished institutional ones.

## Archetypes

Useful archetypes:
- John Peel
- Rodney Bingenheimer
- pirate radio
- niche scene magazines
- tastemaker DJs
- college/community radio
- creator platforms with real editorial selection
- marketplaces or communities where movement shows up before journalists notice

These matter because attention there is a clue, not because they are universally respected.

## Selection Questions

Before adding a source, ask:

1. Is it early?
Does it surface movement before broader validation?

2. Is it selective?
Does appearance there mean something, or is it just a feed of everything?

3. Is it close to the scene?
Is it embedded in creators, communities, venues, critics, collectors, or other live cultural infrastructure?

4. Is it consequential?
Does attention from this source help signal, validate, or accelerate cultural movement?

5. Does it help corroboration?
Can it create meaningful independent crosses with the existing network?

If the answer is mostly no, it should not be in Malcome’s core path.

## Anti-Drift Rules

Do not add a source because it is:
- easy to parse
- well-known
- broadly prestigious
- high-volume
- adjacent to culture

Add it because it helps Malcome get earlier, sharper, and more trustworthy.
